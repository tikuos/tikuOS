TikuKits Libraries

TikuKits is a collection of 46 reusable sub-modules across 8 domains, designed for ultra-low-power microcontrollers. Everything is statically allocated, uses fixed-point math (no FPU required), and follows TikuOS naming conventions.

Naming Convention

Identifier Pattern Example
Functions tiku_kits_<library>_<action>() tiku_kits_sensor_mcp9808_read()
Types tiku_kits_<library>_name_t tiku_kits_sensor_temp_t
Constants TIKU_KITS_<LIBRARY>_NAME TIKU_KITS_SENSOR_OK
Header guards TIKU_KITS_<LIBRARY>_H_ TIKU_KITS_MATRIX_H_
Files tiku_kits_<library>[_detail].h/.c tiku_kits_sensor_ds18b20.c

Return codes are defined in each top-level common header and shared by all sub-modules beneath them. Individual sub-modules do not define their own return codes.

Maths

Module Description
matrix Linear algebra: multiply, transpose, add, identity, determinant, inverse. Fixed-point Q8 elements. Configurable max size via TIKU_KITS_MATRIX_MAX_SIZE.
statistics Online statistics: Welford's running mean/variance, EWMA (exponentially weighted moving average), min/max tracking.
distance Distance metrics: Manhattan (L1), Euclidean (L2), cosine similarity, Hamming distance. Used by ML classifiers.

ML (Machine Learning)

Seven machine learning algorithms, all using fixed-point Q8 arithmetic and zero dynamic allocation. No FPU required.

Module Description
linreg Linear regression (least squares fit)

dtree Decision tree classifier (static tree structure)
knn K-nearest neighbors (configurable K, uses distance module)
nbayes Naive Bayes classifier (Gaussian features)
linsvm Linear support vector machine
tnn Tiny neural network (feedforward, fixed-point weights)

Quick Code Examples

Matrix Multiply

#include "tikukits/maths/linear_algebra/tiku_kits_matrix.h"

tiku_kits_matrix_t A, B, C;
tiku_kits_matrix_elem_t a_data[4] = {256, 0, 0, 256};   /* Q8: identity */
tiku_kits_matrix_elem_t b_data[4] = {512, 256, 128, 64}; /* Q8 values */
tiku_kits_matrix_elem_t c_data[4];

tiku_kits_matrix_init(&A, a_data, 2, 2);
tiku_kits_matrix_init(&B, b_data, 2, 2);
tiku_kits_matrix_init(&C, c_data, 2, 2);

int rc = tiku_kits_matrix_multiply(&A, &B, &C);
/* rc == TIKU_KITS_MATHS_OK, C contains A * B */

ML Prediction (KNN)

#include "tikukits/ml/tiku_kits_ml_knn.h"

tiku_kits_ml_knn_t model;
tiku_kits_ml_knn_init(&model, features, labels, n_samples,
                      n_features, k);

uint8_t predicted_class;
int rc = tiku_kits_ml_knn_predict(&model, test_sample,
                                   &predicted_class);
/* rc == TIKU_KITS_ML_OK */

Convention for new sub-modules: Place files in the appropriate category subdirectory, include the parent common header via #include "../tiku_kits_<kit>.h", use shared return codes from the common header, and keep module-specific configuration macros in the module header.
